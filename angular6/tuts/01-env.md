# Environment Setup
* Yêu cầu cài đặt môi trường
    * Nodejs
    * Npm
    * Angular CLI

## Nodejs
* Kiểm tra nodejs version
```
$ node -v
```
* Kiểm tra npm version
```
$ npm -v
```

# Angular CLI
* Cài đặt angular cli
```
$ npm install -g @angular/cli
```
* Kiểm tra angular version
```
$ ng -v
```

# Example
* Create project with commands
```
$ ng new Angular6App
```
* Run project
```
$ ng serve
# open port default 4200
```
