# RabbitQM cluster with Docker
* Run app
```
# chmod +x cluster-entrypoint.sh
# docker-compose up
```
* RabbitMQ
```
* http://localhost:15672
* guest/guest
```
* HAProxy
```
* http://localhost:1936
```
